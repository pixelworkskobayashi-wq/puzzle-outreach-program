# Puzzle 出前授業ポータルサイト — 設計仕様書

**作成日**: 2026-04-02  
**プロジェクト名**: puzzle-outreach-program  
**ステータス**: 設計確定・実装計画待ち

---

## 概要

北海道建築士会札幌支部が実施する小中学校向け出前授業（間取り作成アプリを使って住まいについて考え学ぶ）の情報発信と申し込み受付を行うポータルサイト。プロトタイプとして関係者（教育委員会・自治体・学校・建築士会）へのプレゼンに使える完成度を目指す。

---

## ターゲットユーザー

- 自治体の教育委員会
- 自治体担当者
- 小中学校の教員・管理職
- 北海道建築士会・関連団体

## サイト構成

### ワンページ構成（ページ遷移なし）

全セクションを1ページに縦積みし、スクロールで閲覧できる。

```
固定ナビゲーション（常時表示）
│
├── HERO
├── ABOUT
├── PROGRAM
├── RESULTS
└── APPLY（申し込みフォーム）
│
FOOTER
```

---

## 各セクション詳細

### 固定ナビゲーション
- 左: ロゴ / サイト名（PUZZLE など）
- 右: ABOUT・PROGRAM・RESULTS・APPLY（各セクションへスムーズスクロール）
- スクロール時も常に上部に固定表示

### HERO
- フルスクリーン（またはそれに近い高さ）
- キャッチコピー（例：「建築のプロが学校へ。子どもたちに空間をデザインする体験を。」）
- CTAボタン2つ：「出前授業を申し込む →」（APPLYへスクロール）・「授業内容を見る」（PROGRAMへスクロール）
- 背景: 授業風景の写真（手元に素材あり）

### ABOUT
- 出前授業の目的・概要の説明（メイン）
- 主催として北海道建築士会札幌支部を簡潔に紹介（サブ）

### PROGRAM
- 授業プログラム1種類を詳細紹介
- 内容: 間取り作成アプリを使って住まいについて考え学ぶ
- 対象学年・授業時間・定員・実施形式などを明記
- **ネットワーク環境必須**である旨を明記（申し込み前の重要事項として）
- アプリ自体の公開・紹介はなし（限定公開のため）

### RESULTS
- 実績数字（実施校数・累計参加人数など）を視覚的に表示
- 授業中・授業後の写真ギャラリー
- 学校・先生からのコメント・感想

### APPLY（申し込みフォーム）
最下部に配置。送信後はページ内でサンクスメッセージに切り替え（ページ遷移なし）。

**フォーム項目:**

| 項目 | 必須 |
|------|------|
| 学校名 | ✓ |
| 担当者名 | ✓ |
| 連絡先メールアドレス | ✓ |
| 連絡先電話番号 | ✓ |
| 希望日程（第1候補） | ✓ |
| 希望日程（第2候補） | ✓ |
| 対象学年 | ✓ |
| 参加人数 | ✓ |
| 実施場所（教室・体育館など） | ✓ |
| ネットワーク環境（Wi-Fi有無・回線種別） | ✓ |
| その他ご要望・備考 | 任意 |

**申し込みフロー:**
1. フォーム送信
2. 担当者にWeb3Formsでメール通知
3. 担当者が内容確認・日程調整
4. 担当者から申込者へメールで承認/否認を返信

### トップへ戻るボタン
- 一定量スクロールすると画面右下または左下に出現
- クリックでページ最上部へスムーズスクロール

### FOOTER
- 主催: 北海道建築士会札幌支部
- コピーライト

## デザイン方針

- **トーン**: 公的機関にも学校にも馴染む中間トーン（信頼感＋親しみやすさ）

- company-hp のCSS設計（デザイントークン・フェードイン・レスポンシブ）を流用・アレンジ
